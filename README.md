<a name="prerequisites"></a>
## Prerequisites

You must be a macOS user and install Xode from the App Store.

### Install Node (NPM)

#### Linux

#### macOS

##### Brew

```
brew install node
```

### Install Ionic

```
npm i -g ionic
```

### Install Cordova

```
npm i -g cordova
```

<a name="start_app_locally"></a>
## Start Applikation Locally

```
ionic serve --address 127.0.0.1 --port 8100
```
